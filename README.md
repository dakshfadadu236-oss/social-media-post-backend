# social-media-post-backend
REST API built with Node.js, Express, MongoDB, Multer and ImageKit for image upload and post creation.

# Node.js Post Upload API

Backend API built with:
- Node.js
- Express.js
- MongoDB
- Multer
- ImageKit

## Features
- Upload image
- Create posts
- Fetch posts
